Guía de Instalación de Tailwind CSS

Sigue estos pasos para instalar Tailwind CSS en tu nuevo proyecto.

Paso 1: Crear un Nuevo Proyecto

Si aún no tienes un proyecto, crea uno nuevo utilizando tu gestor de paquetes favorito, como npm o bun. 

mkdir mi-proyecto
cd mi-proyecto

Paso 2: Instalar Tailwind CSS

Instala Tailwind CSS como una dependencia de desarrollo utilizando npm. 

npm install -D tailwindcss

A continuación, ejecuta el siguiente comando para generar el archivo de configuración de Tailwind: 

npx tailwindcss init

Paso 3: Configurar Tailwind

Abre el archivo tailwind.config.js y edita el contenido del array content para asegurarte de que Tailwind pueda escanear tus archivos HTML y CSS. 

module.exports = {
  content: [
    "./index.html",
    "./src/**/*.{html,js}"
  ],
  theme: {
    extend: {},
  },
  plugins: [],
}

Paso 4: Crear el Archivo CSS

Crea un archivo CSS, por ejemplo styles.css, y agrega las directivas de Tailwind para incluir los estilos básicos, componentes y utilidades. 

@tailwind base;
@tailwind components;
@tailwind utilities;

Paso 5: Agregar el Archivo CSS a tu Proyecto

Agrega el archivo CSS a tu proyecto en el archivo index.html dentro de la etiqueta <head>. 

<link href="./output.css" rel="stylesheet">

Paso 6: Construir y Visualizar

Usa el siguiente comando para iniciar el proceso de construcción y ver tus cambios en tiempo real. 

npx tailwindcss -i ./src/styles.css -o ./src/output.css --watch

Tambien puedes agregar un script en el package.json para correrlo con npm run tail. 

"scripts": {
  "tail": "npx tailwindcss -i ./src/styles.css -o ./src/output.css --watch"
 }

Ahora, abre tu index.html en el navegador para ver los resultados.

Paso 7: Instalar Prettier y el Plugin de Tailwind CSS

Para asegurarte de que tu código CSS esté bien formateado y ordenado, puedes integrar Prettier con el plugin de Tailwind CSS.

Primero, instala Prettier y el plugin de Tailwind CSS como dependencias de desarrollo: 

npm install -D prettier prettier-plugin-tailwindcss

Luego, crea un archivo de configuración de Prettier en la raíz de tu proyecto llamado .prettierrc o edita el existente, agregando lo siguiente: 

{
  "plugins": ["prettier-plugin-tailwindcss"]
}

Ahora Prettier ordenará las clases de Tailwind CSS automáticamente cuando formatees tus archivos. Puedes ejecutarlo manualmente con: 

npx prettier --write .
